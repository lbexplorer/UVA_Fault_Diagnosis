# 实验结果索引

本目录包含UAV故障诊断实验的结果。每个子目录对应一个实验配置。

## 实验列表

### esf_csmlp_v2
- **日期**: 2026-04-10
- **任务**: BASiC stage-2 六类传感器故障分类 (GPS, RC, Accelerometer, Gyroscope, Compass, Barometer)
- **数据集**: 60个故障飞行样本
- **模型**: MLP分类器 (隐藏层: 256-128, dropout=0.3)
- **特征**: 精选特征 (curated) + 增强统计量 (enhanced stats), 特征维度: 663
- **窗口大小**: 1.0秒
- **训练配置**:
  - 损失函数: 加权交叉熵 (weighted_ce)
  - 学习率: 0.001
  - 批大小: 16
  - 轮数: 120
  - 交叉验证: 10折分层随机分割 (test_size=0.3)
- **结果** (平均 ± 标准差):
  - 准确率: 42.78% ± 9.31%
  - 宏平均精确率: 45.85% ± 10.33%
  - 宏平均召回率: 42.78% ± 9.31%
  - 宏平均F1分数: 41.21% ± 9.73%

**文件说明**:
- `summary.json`: 完整实验总结
- `split_metrics.csv`: 每次分割的指标
- `confusion_matrix_mean.csv`: 平均混淆矩阵
- `sample_anchor_check.csv`: 样本锚点信息
- `feature_names.csv`: 使用的特征名称
- `features_used.json`: 特征配置
- `run_config.json`: 运行参数
